# Data Analysis Projects

A collection of exploratory data analysis (EDA) and SQL analytics projects across sports, real estate, and public-health datasets.

## Projects

| Project | File | Stack | Summary |
|---|---|---|---|
| Airbnb NYC 2019 EDA | [`Airbnb NYC 2019 EDA Project.ipynb`](./Airbnb%20NYC%202019%20EDA%20Project.ipynb) | pandas, matplotlib, seaborn | Pricing, availability, and neighbourhood trends across NYC listings. |
| FIFA 19 Dataset | [`FIFA 19 Dataset Project.ipynb`](./FIFA%2019%20Dataset%20Project.ipynb) | pandas, matplotlib, seaborn | Player attributes, value/wage distributions, and positional analysis. |
| IPL 2022 Analysis | [`ipl_2022.sql`](./ipl_2022.sql), [`ipl_2022_project.sql`](./ipl_2022_project.sql) | SQL | Match- and player-level analytics on IPL 2022 results. |
| Global COVID-19 | [`coviddata_world.sql`](./coviddata_world.sql) | SQL | Country-level case/death trend queries over the global COVID dataset. |

## Skills demonstrated

- **EDA in Python** — cleaning, aggregation, and visual storytelling with pandas / matplotlib / seaborn.
- **Analytical SQL** — joins, aggregations, window-style queries against real datasets.
- Communicating findings from messy, real-world data.

## How to use

Open any notebook in Jupyter to view the analysis and visualisations. The `.sql` files can be run against a database loaded with the corresponding source dataset (IPL 2022, global COVID-19).
